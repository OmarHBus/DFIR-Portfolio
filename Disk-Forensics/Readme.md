#Disk-Forensics

En esta carpeta almaceno los análisis relacionados con **forense en disco**, tales como:

- Recuperación de archivos.
- Análisis de sistemas de archivos (NTFS, FAT32, ext4…).
- Investigación de artefactos persistentes.
- MFT, USN Journal, Prefetch, Registry Hives.
- Timeline de actividad del sistema.
- Detección de ejecución de programas.

Aquí incluyo:
- Informes.
- Evidencias procesadas.
- Scripts específicos para análisis en disco.
- Ejercicios prácticos.

*Objetivo:* Mostrar habilidades en adquisición, parsing, correlación y análisis de artefactos estáticos almacenados en el disco.
