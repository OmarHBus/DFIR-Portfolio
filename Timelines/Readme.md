#Timelines

Aquí organizo líneas de tiempo generadas durante análisis forense en disco o memoria.

Incluye:
- Artefactos de tiempo (MFT, Prefetch, Shimcache…)
- Correlación de eventos.
- CSV/JSON procesados.
- Graficado de eventos clave.
- Scripts de timeline automático.

Herramientas usadas:
- plaso / log2timeline
- Timesketch
- mftdump / MFTECmd
- Volatility timeliner

*Objetivo:* Reconstruir la secuencia exacta de eventos para entender el incidente.
