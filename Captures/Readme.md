#Captures

En esta carpeta guardo **todas las capturas de pantalla** generadas durante mis análisis forenses.

Su objetivo es documentar visualmente:
- Evidencias clave.
- Resultados de comandos.
- Pasos relevantes del análisis.
- Anomalías detectadas.

Las capturas nunca incluyen datos sensibles o información personal. Todas las imágenes están pensadas para complementar informes, presentaciones o ejercicios de laboratorio.
