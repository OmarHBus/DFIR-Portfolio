#Memory-Forensics

En esta carpeta documento análisis de **memoria RAM** utilizando herramientas como:

- Volatility 2
- Volatility 3
- Rekall
- Strings
- YARA

Aquí se incluyen:
- Dumps procesados (sin volcar los .raw).
- Scripts de extracción.
- Evidencias relevantes (procesos, conexiones, módulos).
- Informes de análisis.
- Casos prácticos de CTF o ejercicios de laboratorio.

Ejemplos de análisis:
- Procesos ocultos.
- Código inyectado.
- Artefactos residuales.
- Recuperación de imágenes, strings o archivos desde memoria.

*Objetivo:* Demostrar dominio en el análisis volátil para IR y DFIR avanzado.
