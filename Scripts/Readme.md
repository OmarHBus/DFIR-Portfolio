#Scripts

Carpeta destinada a scripts que desarrollo para automatizar tareas de análisis, tales como:

- Parsing de logs.
- Decodificación de cadenas.
- Búsqueda de indicadores.
- Extracción de artefactos.
- Automatización de Volatility.
- Herramientas auxiliares para DFIR.

Suelen estar escritos en:
- Python
- PowerShell
- Bash

*Objetivo:* Centralizar utilidades creadas durante los análisis y facilitar su reutilización.
